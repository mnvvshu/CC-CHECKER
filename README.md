## 📌 Overview

A multi-threaded Python CLI application designed to demonstrate:

* Request handling
* Input parsing & validation
* Proxy formatting
* Concurrent task execution

Built with a focus on **performance**, **clean terminal UI**, and **structured workflow**.

---

## 🚀 Features

* ⚡ Multi-threaded processing (ThreadPoolExecutor)
* 🌐 HTTP session handling with proxy support
* 🎨 Colored CLI interface (ANSI formatting)
* 📂 Automatic file handling and result logging
* 🔍 Smart input parsing and normalization
* 🧵 Thread-safe file writing

---

## 🛠️ Tech Stack

* Python 3
* requests
* concurrent.futures
* threading
* standard libraries
